// About
KhiemHuy Dev Arcade
Một dự án cá nhân kết hợp game, thuật toán và developer tools.

🎯 Về dự án

KhiemHuy Dev Arcade là một nền tảng game giáo dục, được xây dựng để giúp developers hiểu sâu hơn về cách các thuật toán hoạt động trong thực tế — thông qua việc chơi game và quan sát trực quan.

Toàn bộ logic game được port từ codebase C# gốc, đã được refactor theo chuẩn senior developer: tách biệt concerns, single responsibility principle, và clean architecture.

⚙️ Kiến trúc hệ thống

  • →
    RulesEngine — Dictionary-based rule lookup O(1), đảo ngược kết quả không cần duplicate rules.
  • →
    MarkovChain — Ma trận chuyển tiếp xác suất để dự đoán nước đi tiếp theo của người chơi.
  • →
    PatternDetector — Sliding window 3-grams để phát hiện lặp lại pattern.
  • →
    HistoryStore — localStorage với JSON serialization, tương tự JSON-lines trong C# gốc.
  • →
    AlgoVisualizer — Generator-based step-by-step execution với async delay control.

📚 Thuật toán được dùng

  • AI
    Markov Chain (bậc 1) — CPU học từ transition matrix P(next | current). Mỗi ván chơi cập nhật ma trận.
  • AI
    3-gram Pattern Matching — Tìm subsequence giống nhau trong lịch sử để dự đoán nước tiếp theo.
  • SORT
    Bubble / Insertion / Selection / Quick / Merge Sort — Visualizer step-by-step với async generators.
  • CONV
    AI Code Translation — Dùng LLM để chuyển đổi cú pháp giữa các ngôn ngữ lập trình.
